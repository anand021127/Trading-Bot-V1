Frontend (Vite + React)

Commands:

```bash
# install dependencies
npm install

# dev server
npm run dev

# build
npm run build

# preview production build
npm run preview
```

Environment variables
- Create `.env` in `frontend/` with `VITE_BACKEND_URL` if your backend is hosted remotely.

Example:

```
VITE_BACKEND_URL=https://upstox-bot-api.onrender.com
```
